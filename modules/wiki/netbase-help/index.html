Welcome to NetBase

NetBase is a collaboration platform. It is very light-weight, modular and KISS: keep-it-simple & stupid.
Goal: to have a central place on the network where a team can share files, information and progress.

Visit our GitHub at: www.github.com/G3CK0-NL/netbase.



The following services are available:
  • File share
    Samba: (anonymous access)
    Web:

  • Collaboration (Zulip)
    (first start takes some time)
    At first use, create a new organisation/admin:
    Then allow mortal users to sign up through the main page:
    • Go to settings
    • Select 'Manage organization'
    • On the left, click 'Organization permissions'
    • Find the section 'Joining the organization'
    • Set 'Are invitations required for joining the organization' to No
    • Set 'Restrict email domains of new users?' to No
    • Click 'Save changes'

  • Videoconferencing (Jitsi)


  • Wiki (DocuWiki)
    (credentials: admin:admin)

  • Live editing (Etherpad)

    Admin login: (credentials: admin:admin)

  • GitLab


  • CyberChef


  • Draw.io
